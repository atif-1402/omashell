## ‚ú¶ Features

### ‚ĖćBar
> A minimal bar with clean workspace styling, Omarchy status modules, and performance stats. Theme‚ÄĎaware, lightweight, and designed to stay out of the way while still being informative.

- **Omarchy button** ‚ÄĒ opens the main launcher
- **Workspaces** ‚ÄĒ shows your workspace list and highlights the active one
- **Media** ‚ÄĒ displays the current track and opens the mini player card on hover *(takes colors from album cover art for a more aesthetic look)*
- **System stats** ‚ÄĒ quick glance CPU and RAM, shows a progress bar on hover
- **Tray** ‚ÄĒ for applications only
- **Notification & Control Center** ‚ÄĒ access notifications, volume sliders, DND, Wi-Fi and Bluetooth manager
- **Clock** ‚ÄĒ shows time and date; hover reveals a calendar
- **Notification indicator** ‚ÄĒ lets you know when something arrives
- **Settings** ‚ÄĒ switch bar position, bar style, launcher logo, and workspace style live from the UI
- Includes Omarchy idle locking and more

---

### ‚ĖćDock
> Fast minimal and functional dock.

- Main button like launcher and taskmanger always pinned
- Smart Auto hide setting
- Right click to pin app to dock

---

### ‚ĖćLauncher
> Fast, minimal, and keyboard‚ÄĎfirst. Opens instantly, filters as you type, and stays clean and distraction‚ÄĎfree.

- Instant search across all sub-menus (Global Search)
- Inspired by the Omarchy Walker menu
- Quick sections for utilities and custom menus with dedicated keybinds
- Flatpak application support

### ‚ĖćTheme & Background Picker
> A fresh interface for switching themes and wallpapers on the fly.

- Dedicated keybinds for both
- One-row scrollable layout
- Keyboard and mouse navigation

---

> A new-style OSD for volume and media with a fluid visualizer running in the background.

- Dedicated keybinds for each OSD
- Media OSD shows progress bar and cover art

### ‚ĖćMedia Card

- Adapts colors from album cover art

- Image previews in Clipboard with a clear-all button

## ūüď¶ Installation

> [!WARNING]
> This setup modifies your Hyprland configuration. **Backup your configs before continuing.**
> Do **not** blindly overwrite configs unless you know what you are doing.

### 1 ‚ÄĒ Install dependencies

```bash
sudo pacman -S --needed quickshell cava cliphist wl-clipboard imagemagick jq
```

### 2 ‚ÄĒ Clone the repository

```bash
git clone https://github.com/atif-1402/anomshell.git
cd anomshell
```

### 3 ‚ÄĒ Backup existing config *(recommended)*

```bash
cp -r ~/.config/quickshell ~/.config/quickshell.backup 2>/dev/null
cp ~/.config/hypr/bindings.conf ~/.config/hypr/bindings.conf.backup 2>/dev/null
```

### 4 ‚ÄĒ Install Quickshell config

```bash
cp -r config/quickshell ~/.config/
```

### 5 ‚ÄĒ Install keybinds *(manual merge recommended)*

Open `config/hypr/bindings.conf`, copy the Quickshell binds, and **manually paste them at the end of** `~/.config/hypr/bindings.conf`.

> This avoids breaking your existing keybind setup.

### 6 ‚ÄĒ Install helper scripts

```bash
cp local/bin/* ~/.local/bin/
chmod +x ~/.local/bin/qs.sh
chmod +x ~/.local/bin/wb-kill.sh
```

### 7 ‚ÄĒ Start Quickshell

```bash
~/.local/bin/qs.sh
```

or use the keybind: `SUPER + SHIFT + Q`

### 8 ‚ÄĒ Start Quickshell at startup *(optional)*

```bash
cat >> ./.config/hypr/autostart.conf << 'EOF'
exec-once = ~/.local/bin/wb-kill.sh
exec-once = ~/.local/bin/qs.sh
EOF
```

---

### ūüďĚ Notes

- Ensure `~/.local/bin` is in your `$PATH`
- Restart Hyprland if binds don't work
- Manual config merging is intentional ‚ÄĒ it prevents overwriting your existing setup
- Tested on a personal Omarchy setup
